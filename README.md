O Replace é uma plataforma web desenvolvida com o objetivo de reduzir o desperdício de alimentos e ajudar mercados a diminuírem prejuízos causados por produtos próximos da validade.

A proposta conecta mercados e consumidores através de um sistema inteligente de descontos, permitindo que alimentos ainda em boas condições sejam vendidos por preços reduzidos antes do vencimento.

O sistema oferece uma experiência moderna e intuitiva, permitindo que mercados cadastrem produtos, definam descontos e acompanhem reservas em tempo real.

Além de gerar economia para os consumidores, o Replace contribui diretamente para a sustentabilidade, incentivando o reaproveitamento de alimentos e reduzindo impactos ambientais.

Principais funcionalidades:
- Cadastro de produtos
- Sistema de descontos automáticos
- Visualização de ofertas
- Interface moderna e responsiva

Tecnologias utilizadas:
- React JSX
- Node.js
- Express
- PostgreSQL
- CSS moderno

Objetivo do projeto:
Transformar perdas em oportunidades através da tecnologia, conectando sustentabilidade, economia e inovação em uma única plataforma.
